# 🎓 Task 2 – VaultofCodes Verification Portal Redesign

This repository contains my submission for **Task 2** of **Assignment 3 – Design & Code Challenge** under the **VaultofCodes Web Development Internship Program**.

The objective of this task was to redesign the existing **VaultofCodes Certificate Verification Portal** while preserving its original functionality. The project focuses on improving the user interface, responsiveness, and overall user experience using modern frontend development techniques.

---

## ✨ Features

- Premium Light Theme
- Glassmorphism UI Design
- Responsive Layout
- Modern Verification Form
- Student Verification Dashboard
- Assignment Progress Tracker
- Verified Student Badge
- Certificate Download Button
- Animated Loading Spinner
- Toast Notifications
- Smooth Hover Effects
- Modern Typography
- Mobile-Friendly Design

---
